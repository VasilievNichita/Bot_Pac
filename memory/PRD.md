# PCB Quiz Telegram Bot - PRD

## Original Problem Statement
Создать Telegram бота с тестами на основе 4 PowerPoint презентаций о печатных платах (PCB). Банк вопросов 150-180 вопросов с вариантами a,b,c,d. Личная статистика, перемешивание вопросов, показ правильного ответа после ответа.

## User Persona
- Студенты изучающие проектирование печатных плат
- Специалисты желающие проверить свои знания по PCB

## Core Requirements
- ✅ Telegram бот с quiz функционалом
- ✅ 180 вопросов по темам PCB
- ✅ 4 варианта ответа (a,b,c,d)
- ✅ Личная статистика (% правильных)
- ✅ Показ правильного ответа сразу после ответа
- ✅ Перемешивание и бесконечное повторение вопросов

## What's Been Implemented (2026-03-11)
1. **JSON банк вопросов** - `/app/backend/questions.json` (180 вопросов)
2. **Telegram Bot** - `/app/backend/telegram_bot.py`
3. **MongoDB интеграция** - хранение статистики пользователей
4. **Supervisor конфигурация** - автозапуск бота

## Architecture
- **Bot Framework**: python-telegram-bot 21.0
- **Database**: MongoDB (collections: quiz_stats, current_questions)
- **Deployment**: Supervisor managed process

## Question Topics Covered
1. История печатных плат (Хэнсон, Эдисон, Эйслер)
2. Типы печатных плат (однослойные, двуслойные, многослойные, гибкие)
3. Технологии производства (субтрактивная, аддитивная, синтеза)
4. Материалы (FR-1 до FR-5, G10, G11, керамика)
5. Проектирование и расчёт параметров
6. Проверка и контроль качества
